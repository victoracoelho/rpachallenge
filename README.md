# Execução:

Arquivo 'challenge.xlsx' deve estar no diretório padrão onde o python é executado. Todas as saídas do código estarão no mesmo caminho.

Para descobrir o caminho, executar no python:

import os  
print(os.getcwd())
  
  
  
Atenção: O código contém as dependências para Chromedriver, caso seja necessário.
